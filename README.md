Task Management System разворачивается с помощью docker-compose, все доступные эндпоинты можно посмотреть после запуска программы по ссылке "http://localhost:8080/swagger-ui/index.html#".
Чтобы запустить приложение на компьютере должен быть установлен Docker. 
Для запуска в командной строке откройте папку taskManager и введите команду "docker-compose up".
Также в проекте присутствует файл с тестами postman, который можно проводить после запуска. Со временем колличество тестов будет увеличиваться
